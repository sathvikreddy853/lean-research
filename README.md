# Lean Research

This repository contains formal proofs of mathematical theorems using the **Lean 4 theorem prover** and **mathlib**.

## Overview

The goal of this repository is to:
- formalize mathematical theorems in Lean
- understand how proofs translate into a formal system
- build a structured collection of verified results

Currently, the repository contains proofs for a small number of theorems, with more to be added over time.

## Work Included

### Mantel's Theorem

The first formalized result in this repository is **Mantel's Theorem** from graph theory:

> A triangle-free graph on  $n$ vertices has at most $\frac{n^2}{4}$ edges.

The implementation can be found in: [mantel/mantel.lean](https://github.com/sathvikreddy853/lean-research/tree/main/mantel)
